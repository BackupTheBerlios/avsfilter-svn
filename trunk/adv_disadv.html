About

This filter is used for load avisynth script as native avidemux2 video filter. Filter supports external binary Windows dll in avisynth script (thanks wine), several AvsFilter instance with different avisynth script and many other features.

Main advantages of AvsFilter :

  • work as native avidemux2 filter;
  • supports several AvsFilter instances with various avisynth scripts in avidemux2 video filter list;
  • supports binary windows dll in avisynth script are supported (with using wine);
  • supports avisynth script with multiple sources, such as AVISource, etc (instead one main videostream from previous avidemux2 video filter);
  • main video stream parse and decode with native Linux libraries (like ffmpeg/xvid/etc) instead of using Windows dll like ffdshow/xvidenc/etc with avs2yuv/avsproxy;
  • avidemux2 project save/restore work correctly with avsfilter parameters
    •

Disadvantages of AvsFilter :

  • reasonable processing degradation with comparision of VirtualDub + avisynth script in Windows;
  • delay after add filter in avidemux2 filter list (lazy start of wine);
  • work only in Linux (but someone can adopted filter to other OS);
  • need wine installation;
  • manual editing "Makefile" (change pathnames) for compile instead of using ./configure at this stage;
  • complex compile wine part (wine executed application - avsloader.exe and pipe_source.dll), but you can use binary wine apps;
  • no rpm/etc binary modules for avsfilter.so;
  • need very small avisynth script correction;
  • filter supports only video processing in avisynth script, audio not supported;
  • don't store parameters of avsfilter, if use not patched version of avidemux2.
